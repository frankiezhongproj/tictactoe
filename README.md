# Tic-Tac-Toe Game (Python)

## Description
This is a simple Tic-Tac-Toe game made in Python that runs in the command line. Two players take turns placing Xs and Os on a 3x3 board, and the program checks for a winner or a draw.

## Features
- 2-player turn-based gameplay
- 3x3 board using a 2D array
- Checks for win conditions (rows, columns, diagonals)
- Detects when the game is a draw when all spots on the board are filled
- Basic input validation so players can’t pick invalid spots (ex. filled or out of bounds)
- Handles invalid inputs with try and except
## Technologies Used
- Python
